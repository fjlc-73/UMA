Exercise 3 asks us to write a WHILE program that computes the sum of three numbers. My proposal is the following: \begin{lstlisting}
Sum3=(3,s)
s:
   while X2 $\neq$  0 do
      X1 := X1 + 1
      X2 := X2 - 1;
   od
   
   while X3 $\neq$  0 do
      X1 := X1 + 1
      X3 := X3 - 1;
   od
